# Инструкции по изображениям для сайта CineSchedule

В эту папку нужно разместить следующие изображения для сайта:

## Обязательные изображения

### 1. hero-main.png
- **Описание**: Главная панель управления CineSchedule
- **Размер**: Рекомендуется 800x600 или больше
- **Где используется**: Hero секция, главное изображение
- **Примечание**: Должно показывать основной интерфейс с расписанием

### 2. analytics.png
- **Описание**: Скриншот секции аналитики
- **Размер**: Рекомендуется 800x600
- **Где используется**: Раздел "Аналитика и Контроль"
- **Примечание**: Должен показывать графики активности смены и загрузки залов

### 3. ads.png
- **Описание**: Скриншот мониторинга рекламных блоков
- **Размер**: Рекомендуется 800x600
- **Где используется**: Раздел "Мониторинг контента"
- **Примечание**: Должен показывать структуру рекламных блоков (EMBEDED/COMMERCIAL)

### 4. settings.png
- **Описание**: Скриншот настроек приложения
- **Размер**: Рекомендуется 800x600
- **Где используется**: Раздел "Надежность"
- **Примечание**: Должен показывать панель настроек

### 5. schedule-grid.png
- **Описание**: Сетка расписания по залам
- **Размер**: Рекомендуется 1200x800 (большое изображение)
- **Где используется**: Галерея интерфейсов, большой элемент
- **Примечание**: Главное изображение галереи

### 6. releases.png
- **Описание**: База фильмов и релизов
- **Размер**: Рекомендуется 600x400
- **Где используется**: Галерея интерфейсов, маленький элемент
- **Примечание**: Должен показывать список фильмов с постерами

### 7. customization.png
- **Описание**: Настройки персонализации
- **Размер**: Рекомендуется 600x400
- **Где используется**: Галерея интерфейсов, маленький элемент
- **Примечание**: Должен показывать цветовую кодировку

### 8. system-core.png
- **Описание**: Системная информация и обновления
- **Размер**: Рекомендуется 1200x600 (широкое изображение)
- **Где используется**: Галерея интерфейсов, широкий элемент
- **Примечание**: Должен показывать информацию о версии и системе

## Рекомендации

- Формат: PNG или JPG (PNG предпочтительнее для скриншотов интерфейсов)
- Оптимизация: Сжимайте изображения для быстрой загрузки, но сохраняйте читаемость текста
- Стиль: Используйте темную тему интерфейса приложения
- Разрешение: Минимум 1200px по ширине для больших изображений, 800px для обычных

## Как заменить изображения

Просто замените файлы в этой папке на свои скриншоты с теми же именами. Сайт автоматически подгрузит новые изображения.
